Initial release of ARGUS — Real-time Global OSINT & Intelligence Dashboard.

**Intelligence & Analysis**
- 3D CesiumJS globe with real-time incident markers, heatmaps, animated arcs, and multi-layer overlays
- 50+ open-source feed aggregation across CONFLICT, CYBER, INTEL, and FINANCE domains
- AI-powered summarization and daily briefings via Ollama, OpenAI, or custom endpoints
- Smart alert system with geofence triggers, keyword matching, and desktop notifications
- Cyber threat intelligence with CVE tracking, ransomware feeds, and IoC extraction

**Tracking & Monitoring**
- Real-time flight tracking (OpenSky Network) with military aircraft filtering
- Maritime vessel tracking with route visualization
- Earthquake monitoring (USGS) and disaster tracking (GDACS)
- Nuclear facility monitoring, pandemic surveillance, and weather alerts
- VIP social media monitoring (X/Twitter) and Telegram channel aggregation

**Media & Live TV**
- Multi-channel live TV grid (2×2 / 3×3 / 4×4) with 40+ preset news channels
- YouTube live stream resolution with optimized iframe embedding
- HLS.js player for direct m3u8 streams with 480p quality cap
- Template system for saving and loading channel layouts

**Architecture**
- Electron 41 + React 19 + TypeScript + electron-vite
- SQLite (better-sqlite3) offline-first incident caching
- Multi-process IPC with cascade data refresh pipeline
- Zustand state management with persistence
- Content Security Policy hardened for production
